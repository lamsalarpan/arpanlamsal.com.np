Flappy Bird

A browser-based recreation of the classic Flappy Bird game, built with plain HTML, CSS, and JavaScript. The bird is controlled with a key press or tap, flying between gaps in scrolling obstacles — the goal is simply to survive as long as possible without hitting one. It was a fun, quick project for practicing game logic like collision detection, gravity, and animation timing using nothing but vanilla JS.
